# Meldingsbesluit Toetshulp

Deze tool toetst of een handeling *meldingsplichtig* lijkt onder Hoofdstuk 2 van het Meldingsbesluit VCRO.

## Gebruik
1. Maak een GitHub-repo.
2. Upload alle bestanden.
3. Ga naar **Settings → Pages → Build from branch → main → /root**.
4. Jouw tool staat nu online.

## Bestanden
- `index.html` – UI
- `style.css` – Styling
- `schema-loader.js` – Laadt schema
- `app.js` – Engine (skip-logic + routes A–H)
- `schema.json` – ALLE vragen, ALLE drempels
- `README.md` – Uitleg
